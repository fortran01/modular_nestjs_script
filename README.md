## Prerequisites

- Node.js

## Installation

- Clone the repository
- Install the dependencies

```bash
npm install
```

## Seeding the database

```bash
npm run seed
```

## Running the application

- Start the application

```bash
npm run build
npm run start
```

The application will be running on [http://localhost:3000](http://localhost:3000)
